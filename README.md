## Запуск

Для запуска необходимо выполнить следующую команду в корне репозитория:

```bash
dotnet run --project src/BooleanSearcher.csproj
```

> Для работы программы необходима папка с леммами по каждой странице `lemmas_per_doc` из 2 задания.
> Также необходим файл с индексами `index.txt` из задания 1.

## Описание

Приложение строит инвертированный индекс по токенам документов и выполняет булев поиск по пользовательскому запросу.

Работа программы состоит из двух шагов:

1) Построение инвертированного индекса

- Для каждого документа читается список токенов из папки tokens_per_doc/.

- Для каждого термина определяется, в каких документах он встречается.

- Формируется инвертированный индекс вида: `термин -> список документов`.

- Построенный индекс сохраняется в файл `inverted_index.txt`.

2) Булев поиск

После построения индекса программа ожидает ввод запроса с консоли.

Поддерживаются операторы:

- `AND` — логическое И

- `OR` — логическое ИЛИ

- `NOT` — логическое НЕ

- круглые скобки для задания приоритета операций

> Запрос вводится строкой с консоли.

Для завершения работы необходимо ввести:

```text
exit
```

## Результат

После выполнения создаётся файл:

- `inverted_index.txt` — инвертированный индекс в формате: `термин: docId1, docId2, docId3`

Во время работы программа выводит в консоль:

- количество найденных документов

- список документов, удовлетворяющих булеву запросу
